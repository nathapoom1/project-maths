# project-maths

Small repository containing a Gauss-Seidel method example script.

Files
- `Gauss-Seidel method.py` — Python script that implements Gauss–Seidel iterations and interactive input for matrix A and vector b.

Quick run (Windows, requires Python 3 and numpy):

1. Install numpy if needed:

   python -m pip install numpy

2. Run the script:

   python "Gauss-Seidel method.py"
